Question-1: What are the different ways to select an element in the DOM?

There are many way to select an element in DOM. Such as getElementByTagName(), getElementById(), getElementsByClassName(), querySelector(), querySelectorAll() etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText and textContent is innerHTML return content with tagname, innerText gives only visible text, textContent gives visible and hidden all text.

Question-3: What is event delegation in the DOM?

Event delegation is process when I add single event listener to parent element and it handle event for all child element. dont need to add event listener for each child separately.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event, like a click, starts at the item I clicked and then bubbles up to its parents, and their parents, and so on right up to the top.

Question-5: How do you create, add, and remove elements using JavaScript?

I create element using document.createElement(), then set inner text. Then adding element using appendChild(), I remove element using removeChild()
